Interactive ROI calculator showing potential automation savings for South African businesses.

== Description ==

The Do.Qix ROI Calculator is an interactive widget that lets prospects estimate their potential savings from business process automation. Five sliders control the inputs (team size, hours, hourly cost, automation efficiency, and error cost), and the results update in real-time.

**Features:**

* Zero dependencies — vanilla JS, no jQuery
* Fully responsive with touch-friendly 36px+ slider thumbs on mobile
* Configurable pricing tiers, slider ranges, CTA, and share button via admin settings
* Conditional asset loading — CSS/JS only on pages with the shortcode
* Themify Builder compatible
* Translation-ready with text domain

**Usage:**

Place the shortcode `[doqix_roi_calculator]` on any page or post.

Configure settings at **Settings > ROI Calculator**.

== Installation ==

1. Upload the `doqix-roi-calculator` folder to `/wp-content/plugins/`
2. Activate the plugin through the Plugins menu
3. Add `[doqix_roi_calculator]` to any page or post
4. (Optional) Adjust settings at Settings > ROI Calculator
