Mobile App

Offline-First Field App

Your field staff don't always have signal. NomadIQ's PWA works fully offline, queuing actions locally and syncing automatically when connectivity returns. Install it directly from the browser.

GPS-Verified Check-In

Staff check in at sites with GPS verification and accuracy display. Select from existing sites or log new locations for admin approval.

Photo Capture

Capture multiple photos per check-in for proof of work, site conditions, or issue documentation. Photos sync with the dashboard automatically.

Material Requests

Field staff request materials during check-in. Requests flow to the dashboard for manager review and fulfilment tracking.

Offline-First

Full offline operation with Workbox service workers. Actions queue locally and background sync handles the upload when signal returns.

Multi-Tenant

Organisation slug-based routing with per-tenant branding (colours, logo). Installable on mobile devices via Add to Home Screen.

Auto-Update

The app notifies staff when updates are available and auto-refreshes at midnight. Sync status indicator keeps everyone informed.

Technical Foundation

Built on Solid Ground

NomadIQ is not a no-code hack. It's a full-stack platform built with modern tools and enterprise-grade security. Here's what's under the hood.

React 19 + Vite 7

Modern frontend with TanStack Query, React Router v7, and Tailwind CSS v4.

Self-Hosted PostgreSQL

Your data stays on SA infrastructure. PostgREST API. 90+ migrations. Row Level Security.

Custom JWT + MFA

TOTP-based multi-factor authentication, recovery codes, rate limiting, and refresh token rotation.

Five-Role RBAC

Super User, Admin, Manager, Staff, Accountant. Each with progressively scoped permissions.
