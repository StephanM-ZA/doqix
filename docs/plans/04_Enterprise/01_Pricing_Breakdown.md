| Item | Amount |
|------|--------|

## Three Sub-Tier Anchors

See [`sub_tiers/`](./sub_tiers/) for full detail per sub-tier.

### Foundation — From R80K setup + R15K/mo
- 3 core workflows
- Standard integrations (common SaaS)
- Email support, 2-business-day first response
- Monthly check-in (15 min)
- 99.0% uptime commitment
- **For:** organisations testing the managed-service approach

### Growth ⭐ *Recommended* — From R180K setup + R35K/mo
- 6–8 workflows
- Priority integrations, including medium-complexity APIs
- Email + WhatsApp support, 8-business-hour first response
- Quarterly business review
- Change-request credit pool (10 hrs/mo)
- 99.5% uptime commitment
- **For:** established mid-market, multi-department rollout

### Partnership — From R400K setup + R65K/mo
- Unlimited workflow coverage (within scope)
- Legacy / on-prem / regulated-industry integrations
- Dedicated solutions engineer
- Email + WhatsApp + phone, 1-hour first response
- Monthly roadmap sessions
- On-call SLA for P1
- SSO, audit trails, role-based access
- 99.9% uptime commitment
- **For:** true enterprise, regulated industries, high-integration complexity

## Costing Levers

| Lever | Typical range |
|-------|---------------|
| Discovery sprint (paid, before quote) | R15,000 – R30,000 (credited against build) |
| Build per workflow | R8,000 – R40,000 |
| Blended build rate | R850 – R1,500/hr (default R1,100) |
| Integration complexity uplift | +0%–40% |
| First-time enterprise risk buffer | +15%–30% |
| Compliance overhead (POPIA / ISO / PCI) | +5%–25% |
| Annual uplift | CPI + up to 3% |
| Change requests (outside retainer) | R5,000 – R25,000 each |
| Multi-year discount (max) | –10% monthly |
| Reference-customer discount | –5% monthly |

## Internal Margin (reference)

| Sub-tier | Monthly | Hard costs | Maintenance | Gross margin |
|----------|---------|------------|-------------|--------------|
| Foundation | R15,000 | ~R300 | ~R1,200 (4hrs) | ~R13,500 (~90%) |
| Growth | R35,000 | ~R500 | ~R3,500 (10hrs) | ~R31,000 (~88%) |
| Partnership | R65,000 | ~R1,000 | ~R15,000 (50hrs + dedicated eng allocation) | ~R49,000 (~75%) |

Target blended engagement margin: **≥ 50%** (see `sales_tooling/Pricing_Model_Reference.md` §7).

## Contract Terms

- **Term:** 12 months minimum
- **Annual uplift:** CPI + up to 3%
- **Termination:** 90 days written notice
- **IP:** Client owns workflows; Do.Qix retains reusable components
- **Data:** Client-hosted or SA cloud (default), per compliance requirements
- **SLA:** Per sub-tier; defined in proposal appendix
