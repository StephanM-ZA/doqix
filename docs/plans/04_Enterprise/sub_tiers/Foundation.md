# Enterprise — Foundation Sub-Tier

**From R80,000 setup + R15,000/mo**

---

## Who it's for

Enterprises testing the managed-service approach before committing deeper. Often a pilot or first-wave engagement to prove the model internally.

---

## What's included

- **3 core workflows** — scoped during discovery
- **Standard integrations** — common SaaS, REST APIs
- **Email support** — 2-business-day first response
- **Monthly check-in** (15 min)
- **Uptime commitment:** 99.0%
- Hosting, monitoring, maintenance for the 3 workflows
- POPIA-compliant handling
- Documentation package

---

## What's excluded (or add-on)

- Priority / on-call SLA → Growth or Partnership
- WhatsApp or phone channels → Growth+
- Legacy / on-prem integrations → Partnership (or scoped add-on)
- SSO / audit trails / RBAC → Partnership (or scoped add-on)
- Dedicated engineer → Partnership
- Quarterly/monthly reviews → Growth+

## Sell it when...

- Client is new to managed automation
- Budget is capped for year 1
- Internal buy-in needs a win before scaling
- 3 workflows meaningfully cover their pilot scope

## Don't sell it when...

- Multi-department rollout from day one
- Compliance / regulated industry
- Client expects priority SLA
- 6+ workflows needed in year 1 → push to Growth
