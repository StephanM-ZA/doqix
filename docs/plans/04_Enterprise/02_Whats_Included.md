# Enterprise — What's Included

**Version:** 1.0 · **Date:** 2026-04-15

---

## Included (All Sub-Tiers)

- **Scoped workflow coverage** — from 3 (Foundation) to unlimited (Partnership)
- **Discovery sprint** (paid separately; credited against build if proceeding)
- **Full implementation** — design, build, test, pilot, rollout
- **Managed hosting** — SA cloud or client-hosted per compliance needs
- **Monitoring & alerting** — proactive, 24-7
- **Maintenance & updates** for contracted workflows
- **POPIA-compliant handling** as standard
- **Training & handover** — initial onboarding per SOW
- **Documentation package** — architecture, runbooks, data flow diagrams
- **SLA-backed support** (response times per sub-tier)
- **Named contact(s)** — varies by sub-tier
- **Ownership** — client owns workflows; no lock-in
- **Month-to-month after first 12** — 90-day termination clause

## Not Included (billed separately, any tier)

| Item | Cost |
|------|------|
| New workflows beyond contracted scope | Quoted per CR, R5K–R25K each |
| Major refactors of existing workflows | Quoted |
| New system integrations | Quoted |
| Training beyond initial onboarding | R1,500/session or bundled |
| After-hours emergency support (non-Partnership) | R500/hr + call-out fee |
| Custom BI dashboards / reporting tools | Custom development quote |
| Bespoke UIs / portals | Custom development quote |
| Third-party API & platform costs | Pass-through (SMS, premium connectors, paid APIs) |

## What Gets Quoted Separately at Sign-Up

The following aren't in the monthly — they're in the setup/one-off SOW:

- Discovery sprint (already paid in most cases, credited)
- Implementation / build of contracted workflows
- Initial training & handover
- Documentation package
- Security review (Partnership default; add-on elsewhere)
- SSO / identity provider integration (Partnership default; add-on elsewhere)
