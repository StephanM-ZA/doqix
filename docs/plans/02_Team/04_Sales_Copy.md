## Objection Handling

| Objection | Response |
|-----------|----------|
| "Why the setup fee?" | "It covers the scoping for three workflows — that's 2–3 proper sessions mapping your processes before we build. Solo skips it because it's one workflow. Team gets the deeper dive." |
| "Can we start with Solo and upgrade?" | "You can, but the math flips fast. Two extra workflows on Solo (+R1,500/mo) is R2,499 — basically Team pricing without the priority support." |
| "What if we only need 2?" | "Team is priced for 3 because that's where most clients land. If you genuinely only need 2, Solo + 1 extra workflow (R1,749/mo) is cheaper. Let's pick the plan that fits, not the biggest." |
| "24-hour support — what if something breaks at 10pm?" | "P1 issues — workflow completely down — we respond same-day during business hours. Out-of-hours emergency support is R500/hr if you need us at 10pm." |
| "Can we add our own workflows later?" | "You own them. Add them yourself if you build the skills, or let us handle it at +R650/mo per workflow." |
