## Conversion Elements (All Pages)

| Element | Placement | Trigger |
|---------|-----------|---------|
| **Header CTA** | All pages, sticky | Always visible |
| **Mobile sticky bar** | Bottom of viewport on mobile | Always visible |
| **Exit-intent popup** | All pages | Mouse moves to browser close/back |
| **Trust badges** | Near every CTA button | Static |
| **Section CTAs** | After sections 4, 6, 8 (Home); after sections 4, 6, 8, 10 (Services) | On scroll |

## Anti-Drop-Off Design Rules

1. CTA visible at all times — header + mobile bar
2. No section longer than one viewport height (mobile)
3. Maximum 2 sections between CTAs
4. Visual element in every section — no text-only blocks
5. Form has 4 required fields, 2 optional — nothing more
6. Page load: under 3 seconds
7. No interstitials, no popups on entry — only exit-intent
